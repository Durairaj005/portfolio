Welcome to my personal portfolio website! This project is a premium, modern, responsive showcase of my skills, projects, and experience as an Artificial Intelligence & Data Science student. ## Features

* **Glassmorphism UI**: Premium dark mode design with sleek animated gradient background shapes.
* **Interactive Sidenav**: Clean vertical navigation dot bar on desktop that transforms into a clear, readable slide-out menu drawer on mobile.
* **Animated Typing Subheading**: Highlights key roles with typing and deleting effects.
* **Modern Skill Badges**: Card grid showing technical proficiencies with clean category tags.
* **Spotlight Highlights**: A specialized section showing key portfolio areas and assessment accomplishments.
* **Machine Learning Feature Card**: A highlighted card detailed with description, technologies, and skills shown for the Machine Learning repository.
* **Responsive Project Showcase**: Grid of interactive cards representing all my latest GitHub repositories, optimized for mobile, tablet, and desktop.
* **Interactive Contact Form**: A functional messaging layout with smooth validation.

## How to Run Locally

Since this is a lightweight frontend project, it does not require any backend compilation or complex framework setup. You can run it locally in seconds:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Durairaj005/portfolio.git
   cd portfolio
   ```

2. **Open in Browser**:
   * Double-click on `index.html` to open it in your default web browser.
   * Or, if using VS Code, right-click `index.html` and select **Open with Live Server** to enable hot-reloading.
